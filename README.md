# Weather App

## 📌 Overview

This is a simple Weather App developed using HTML, CSS, and JavaScript. It fetches real-time weather information for any city using the OpenWeatherMap API and displays the current temperature, weather condition, humidity, wind speed, and weather icon.

## ✨ Features

- Search weather by city name
- Displays current temperature
- Shows weather condition
- Displays humidity
- Displays wind speed
- Weather icon based on current conditions
- Responsive and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## 📂 Project Structure

```
Weather-App/
│── index.html
│── style.css
│── script.js
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Get a free API key from OpenWeatherMap.
3. Open `script.js` and replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your own API key.

4. Save the file.
5. Open `index.html` using Live Server or any web browser.

## 🎯 Purpose

The purpose of this project is to build a weather application that retrieves and displays real-time weather information for any city using a weather API.
